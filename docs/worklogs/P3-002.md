# P3-002 Worklog: Archive Quarter memory workspace

## Status

Complete.

## Goal

Give the expedition a durable, searchable memory surface where discovered sources, interpreted signals, and meeting memos can be found, inspected, compared, filed, and replayed without flattening their different evidentiary roles.

## Intended files

- `packages/archive/src/index.ts`: deterministic local index and faceted search over the authoritative projection and immutable event log.
- `apps/web/src/world-shell/ArchiveWorkspace.tsx`: Archive Quarter search, shelves, inspector, comparison, case-file tray, and replay context.
- `apps/web/src/world-shell/WorldShell.tsx`: workspace routing, keyboard entry, event loading, and synchronized case-file preferences.
- `apps/web/src/world-shell/evidence-preferences.ts`: persistent archive case-file selections.
- `apps/web/src/world-shell/runtime-client.ts`: read-only expedition event retrieval.
- `tests/e2e/archive-workspace.spec.ts`: complete historical-evidence-to-archive journey and visual regression.

## Boundaries

- The archive index reads only the authoritative world projection and immutable expedition events. It does not import or reinterpret fixture content.
- Source, signal, and memo records remain separate archive entry kinds with independent shelves and counts.
- Search is a deterministic in-memory view over already discovered records; it does not append events or change evidence truth.
- Case-file membership is a local organizational preference. Signal case-file entries stay synchronized with the evidence rail's pin state, but neither changes domain status.
- Replay means returning to the exact immutable event sequence where a record entered the expedition. It shows nearby event context instead of re-running side effects.
- Superseded source versions remain independently searchable and inspectable, with an explicit link to the replacing source.

## Acceptance criteria

- Completing Orin's historical mission makes its source and signal discoverable through the Archive Quarter.
- Text, date, place, source-class, and agent filters can be combined and cleared.
- Source, signal, and memo tabs expose truthful per-kind result counts and support left/right arrow navigation.
- A user can stage two records from different shelves and inspect them in a side-by-side comparison.
- Records can be added to a persistent case-file tray; signal selections remain consistent with the signal rail.
- Replay exposes the exact introduction sequence plus nearby immutable events.
- Stale and superseded records remain inspectable rather than disappearing from history.
- The workspace can be entered with `A`, exited with `Escape`, and operated with semantic keyboard controls.

## Results

- Replaced the archive package stub with a typed deterministic index. It builds searchable source, signal, and memo entries from projection entities, preserves supersession chains, associates knowledge holders, derives stable facets, and records the exact introduction event sequence.
- Added tokenized multi-term text search plus inclusive date, place, source-class, and agent filters. Search covers titles, summaries, excerpts, publishers, tags, agents, and linked evidence while retaining strict record-kind boundaries.
- Built the Archive Quarter as a full semantic workspace with a search console, tabbed shelves, record inspector, persistent case-file tray, two-record comparison, and event replay context.
- Kept the first comparison choice as a non-blocking staged state. The full comparison layer opens only after a second record is selected, so the selection workflow remains usable.
- Added `A` and `Escape` workspace shortcuts with editable-control guards, read-only event loading, and projection refresh on entry.
- Extended local evidence preferences with archive IDs and synchronized `signal:<id>` selections with existing signal pins.
- Added archive package unit tests covering combined filters, entry sequences, source history, and superseded/current version visibility.
- Added a browser journey that discovers Orin's historical evidence, combines every filter, uses arrow-key tabs, compares a source with a signal, files evidence, replays its entry, returns to the world, and verifies persistence.
- Added and manually reviewed the 1440 × 900 Archive Quarter baseline. The search console, shelf, inspector, surrounding expedition context, and case-file tray remain legible without obscuring the evidence model.
