## Risks and controls

- **Stream duplication or gaps:** Every connection declares `after=<lastValidatedSequence>`; envelopes are strictly parsed, contiguous event sequences are required, and a snapshot is fetched before accepting a gap.
- **Corrupt client projection:** Stream events never mutate the client projection directly. They trigger an authoritative snapshot catch-up only after envelope validation; the prior projection remains rendered on failure.
- **Reconnect storms:** One client owns one socket, uses bounded exponential backoff with jitter disabled in deterministic tests, and cancels timers on unmount.
- **Test-only fault leakage:** Fault injection remains behind `SIGNAL_ATLAS_E2E=1`, uses explicit testing routes/query values, and cannot alter production behavior.
- **Fallback ambiguity:** A missing local Codex executable is labeled as a scripted fixture fallback, not as a successful local runtime and not as a total application outage.
- **Automated-audit false confidence:** Axe results supplement, rather than replace, keyboard, zoom, forced-colors, screen-reader semantics, and manual focus review.

## Remaining risks

- Automated axe and forced-colors execution currently use Chromium. Cross-browser screen-reader combinations still require a separate manual assistive-technology matrix; the automated results are not presented as universal AT certification.
- The 200% check uses the exact 720 × 450 CSS viewport equivalent of the 1440 × 900 reference desktop and a manual image review. Browser chrome, OS text scaling, and platform-specific font substitution remain external variables.
- Expedition authority and its event log remain process-local. This milestone survives temporary transport loss within one orchestrator process; durable restart recovery belongs to a persistence milestone.
- Phaser remains a large lazily separated production chunk and triggers the existing Vite size advisory. P6-003 owns final load/performance profiling and code-splitting decisions.
- Fault-injection endpoints exist only when `SIGNAL_ATLAS_E2E=1`; production code cannot request an injected disconnect or malformed envelope.
